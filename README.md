SpiderBot 🕷️

What is this?

SpiderBot is a quadruped robot I’m building using an Arduino Nano. It has 4 legs and 12 servos which will allow it to walk, turn, dance, and do other cool movements.

I’ve always been interested in robotics and wanted to build something that actually moves around on its own. This project is also helping me learn more about programming, electronics, and robot design.
